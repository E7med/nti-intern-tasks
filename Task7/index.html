JavaScript Modules Demo

Understanding Modules, Importing, and Exporting in JavaScript

Module Overview

math.js

  • add(a, b) - Addition function
  • subtract(a, b) - Subtraction function
  • pi - Pi constant (3.14159)

greet.js

  • greet(name) - Greeting function

utils.js (Bonus)

  • capitalize(str) - Capitalize first letter

Code Examples

Importing Modules

// Import specific functions and constants
import { add, subtract, pi } from './math.js';
import { greet } from './greet.js';
import { capitalize } from './utils.js';

Using Imported Functions

// Using math functions
const sum = add(5, 3);
const difference = subtract(10, 4);

// Using greeting function
const message = greet("John");

// Using utility function
const capitalized = capitalize("javascript");
